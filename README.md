# Система управління рецептами коктейлів

Консольний Java-додаток для управління рецептами напоїв та коктейлів, що дозволяє зберігати, шукати, редагувати та категоризувати улюблені рецепти.

## Опис системи

Система управління рецептами коктейлів дозволяє:
* Додавати нові рецепти напоїв
* Переглядати всі збережені рецепти
* Шукати рецепти за назвою, категорією та інгредієнтами
* Редагувати існуючі рецепти (назва, інгредієнти, інструкції, категорія)
* Позначати рецепти як улюблені
* Сортувати рецепти за різними критеріями
* Видаляти рецепти

## Структура проекту
├── Main.java               # Головний файл програми з меню та введенням даних
├── Recipe.java             # Клас Recipe для зберігання даних про рецепти
├── RecipeManager.java      # Клас RecipeManager для управління рецептами
├── enums/                  # Пакет з переліченнями
│   ├── RecipeCategory.java # Перелічення категорій напоїв

## Інструкція з використання

### Головне меню програми

При запуску відображається головне меню з наступними опціями:

1. **Додати новий рецепт**
   * Введіть назву напою
   * Введіть інгредієнти через кому
   * Введіть інструкції через кому
   * Виберіть категорію напою
   * Рецепт буде додано з унікальним ID

2. **Показати всі рецепти**
   * Виводить список всіх збережених рецептів з детальною інформацією

3. **Пошук рецепту за назвою**
   * Введіть назву або частину назви рецепту
   * Система відобразить всі відповідні рецепти

4. **Пошук рецепту за категорією**
   * Виберіть категорію з переліку
   * Система відобразить всі рецепти даної категорії

5. **Пошук рецепту за інгредієнтом**
   * Введіть назву інгредієнта
   * Система відобразить всі рецепти, що містять даний інгредієнт

6. **Редагувати назву рецепту**
   * Введіть ID рецепту
   * Введіть нову назву рецепту

7. **Додати інгредієнт до рецепту**
   * Введіть ID рецепту
   * Введіть новий інгредієнт

8. **Видалити інгредієнт з рецепту**
   * Введіть ID рецепту
   * Введіть інгредієнт для видалення

9. **Додати інструкцію до рецепту**
   * Введіть ID рецепту
   * Введіть нову інструкцію

10. **Видалити інструкцію з рецепту**
    * Введіть ID рецепту
    * Введіть інструкцію для видалення

11. **Змінити категорію рецепту**
    * Введіть ID рецепту
    * Виберіть нову категорію

12. **Додати/видалити рецепт з улюблених**
    * Введіть ID рецепту
    * Система змінить статус "улюблений" на протилежний

13. **Сортувати рецепти за назвою**
    * Система відсортує і відобразить рецепти в алфавітному порядку

14. **Сортувати рецепти за кількістю інгредієнтів**
    * Система відсортує і відобразить рецепти за зростанням кількості інгредієнтів

15. **Сортувати рецепти за категорією**
    * Система відсортує і відобразить рецепти за категоріями

16. **Видалити рецепт**
    * Введіть ID рецепту для видалення
    * Рецепт буде повністю видалено з системи

0. **Вихід**
    * Завершення роботи програми
